# Cocktail Collection App

A responsive cocktail recipe app with favorites system. Browse, search, and filter drinks by alcohol type. Saves preferences via localStorage. Built with vanilla JS

## Overview

This application provides an elegant interface for exploring cocktail recipes. It features:

- Searchable grid of cocktails
- Detailed recipe information
- Favorite cocktail tracking
- Persistent storage of user preferences

## Features

- Browse cocktail recipes in card layout
- Mark/unmark favorites with heart icons
- Persistent favorite storage using localStorage
- Search by cocktail name
- Filter by alcohol type (gin, vodka, rum, etc.)
- Responsive design works on all devices
- Detailed recipe information including:
  - Ingredients with precise measurements
  - Preparation instructions
  - Recommended glassware
  - Garnish suggestions

## Installation

1. Clone the repository:

```bash
git clone https://github.com/nvmwhoiam/cocktail-collection-app.git
```

2. Navigate to project directory:

```bash
cd cocktail-collection-app
```

3. Open `index.html` in your web browser or serve it using a local server.
